# قائمة أبو حمودي السوري

قائمة رقمية جاهزة للنشر على GitHub Pages، تعمل بالعربية والإنجليزية، ويمكن تثبيتها على Android وiPhone. بعد أول زيارة متصلة بالإنترنت، تُحفظ ملفات القائمة للعمل دون إنترنت.

## النشر على GitHub Pages

1. أنشئ مستودع GitHub جديدًا.
2. ارفع **محتويات هذا المجلد** إلى جذر المستودع، مع إبقاء أسماء الملفات والمجلدات كما هي.
3. من **Settings → Pages** اختر **Deploy from a branch**.
4. اختر فرع `main` والمجلد `/ (root)` ثم اضغط **Save**.
5. افتح رابط الموقع مرة واحدة بوجود الإنترنت حتى يكتمل التخزين للعمل دون إنترنت.

## التثبيت على الهاتف

- Android / Chrome: افتح القائمة ثم اختر **Install app** أو **Add to Home screen**.
- iPhone / Safari: افتح زر المشاركة ثم اختر **Add to Home Screen**.

## ملاحظة مراجعة

تم نقل الأصناف والأسعار من الصور الموجودة في ملف القائمة المرفق. يُفضّل مراجعة الكتابة والسعر للأسماء التي كانت أقل وضوحًا في الصورة: `جلفراي`، `دجاج مسوي`، `تشريب كمي`، و`أكس ميل`، بالإضافة إلى أسعار قسم السلطات.
